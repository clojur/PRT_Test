# 操作
cmake后将getting_started__GIApplication设为启动项目，在release下运行，键盘按键控制开关如下：
空格：是否显示直接阴影
左ctrl：是否将阴影效果作用到probe
Tab：是否使用动态光源
F1：是否显示直接光照
F2：是否显示Probe
F3：是否显示间接光照
上下左右键：调整光源位置
PageUp/PageDown键：调整光源强弱
